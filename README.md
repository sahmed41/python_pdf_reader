# PDF Reader 

## 📖 Overview
**PDF Reader** is a local desktop application designed for seamless PDF reading, editing, and page manipulation. it is built with a focus on robust core functionality, intuitive user experience, and secure, local-only document processing.

## ✨ Features

### Core Features
- Reading PDF
- Merge multiple PDFs
- Extract specific pages
- Rotate pages
- Delete pages


### Extra Features
- Editing text underlying formatting or font embedding.
- Editing image underlying formatting or font embedding.
- Highlighting, sticky notes, and drawing tools are a must for our review process.
